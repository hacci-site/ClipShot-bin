# ClipShot

**ClipShot** is a powerful screenshot and annotation tool for Linux, designed to enhance your productivity. It allows you to capture any part of your screen and keep it floating as a **"visual sticky note"** or instantly annotate and share it.

![ClipShot Icon](data/clipshot.png)

## ğŸš€ Key Features

- **Floating Sticky Notes**: Keep your screenshots "Always on Top" to use them as references while you code, write, or work.
- **Instant Annotations**: Enhance your captures with pens, markers, lines, arrows, rectangles, and text.
- **Smart Sharing**: Drag images directly from ClipShot into other apps (Slack, Discord, Email, etc.) using the Skitch-style drag handle.
- **Advanced Text Tool**: Insert text with customizable fonts, shadows, and outlines for maximum readability.
- **OCR (Optical Character Recognition)**: Extract text from images instantly using Tesseract OCR.
- **Image Manipulation**: Trim, rotate, and apply mosaic or blur to sensitive information.
- **X11 & Wayland Support**: Works on modern Linux desktops.
