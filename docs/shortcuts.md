# ClipShot Shortcuts & Operations / 操作一覧

## During Capture / スクリーンショット取得中
Operations in the region selection screen after launching. / 起動直後の範囲選択画面での操作。

| Action / アクション | Description / 操作内容 |
|:--- |:--- |
| **Drag / ドラッグ** | Select rectangular region to capture / 矩形範囲を選択してキャプチャ |
| **Click / クリック** | Capture window under cursor (X11 only) / カーソル下のウィンドウをキャプチャ (X11のみ) |
| **Click Background / 背景クリック** | Capture full desktop / デスクトップ全体をキャプチャ |
| `Enter` | Skip capture and open latest image from history / キャプチャをスキップして履歴から最新の画像を開く |
| `Esc` | Cancel capture and exit / キャプチャをキャンセルして終了 |

---

## During Editing & Viewing / 画像編集・表示中
Operations in the main window after capture or opening a file. / キャプチャ後、またはファイルを開いた後の操作。

### Basic Actions / 基本操作
| Key / キー | Action / アクション |
|:--- |:--- |
| `Ctrl + C` | Copy image to clipboard / クリップボードにコピー |
| `Ctrl + S` | Save screenshot / スクリーンショットを保存 |
| `Ctrl + Shift + S` | Save As... / 名前を付けて保存 |
| `Delete` | Delete current file or Discard capture / 現在のファイルを削除、またはキャプチャを破棄 |
| `Ctrl + Z` | Undo / 元に戻す |
| `Ctrl + Y` | Redo / やり直し |
| `Ctrl + O` | Execute OCR / OCR（文字認識）を実行 |
| `Ctrl + P` | Print / 印刷 |
| `Ctrl + ,` | Open Settings / 設定ダイアログを開く |
| `Esc` / `Ctrl + W` / `Alt + F4` | Close window / ウィンドウを閉じる |
| `F1` | About ClipShot / ClipShot について |

### View & Navigation / 表示・ナビゲーション
| Key / キー | Action / アクション |
|:--- |:--- |
| `Ctrl` + `+` / `-` / `0` | Zoom In / Out / Reset (Numpad supported) / 拡大・縮小・リセット（テンキー対応） |
| `F11` or `Ctrl + T` | Toggle Toolbars / ツールバーの表示/非表示を切り替え |
| `←` (Left) / `→` (Right) | Previous / Next image / 前の画像 / 次の画像へ移動 |
| `Page Up` / `Page Down` | Increase / Decrease opacity / 不透明度を上げる / 下げる |

### Tool Selection / ツール選択 (Single Key)
| Key / キー | Tool / ツール |
|:--- |:--- |
| `P` | Pen / ペン |
| `M` | Marker / マーカー |
| `T` | Text / テキスト |
| `L` | Line / 直線 |
| `A` | Arrow / 矢印 |
| `R` | Rectangle / 矩形 |
| `Shift + R` | Filled Rectangle / 塗りつぶし矩形 |
| `C` | Trim / トリミング |
| `O` | Mosaic / モザイク |
| `B` | Blur / ぼかし |

### Parameter Adjustment / パラメーター調整
| Key / キー | Action / アクション |
|:--- |:--- |
| `[` | Decrease line width / 線を細くする |
| `]` | Increase line width / 線を太くする |
